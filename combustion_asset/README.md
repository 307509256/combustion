`combustion_asset`
------------------

Asset importing, exporting and conversion routines.

## Features

- [x] Models
    * Import and export:
        - [x] Combustion model format
    * Import only:
        - [x] External models via Assimp
- [x] Textures
    * Import and export:
        - [x] Combustion texture format
        - [ ] External images via the `image` crate
            - [x] import support
            - [ ] export support
